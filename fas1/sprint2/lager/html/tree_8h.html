typedef int(* comp_func) (void *a, void *b)

A comparison function.

Compares two value and returns a negative value if a is less than b, zero if a and b are equal and a positive value if a is greater than b.

typedef struct tree tree_t

Represents a tree containing key-value pairs.

The tree contains key-value pairs where keys must be comparable and unique.

typedef struct tree_tr tree_tr_t

Respresents a traverser for a tree.

The traverser can be pre-, post-, in- or level-order. The traverser points to one of the nodes in the tree or is 'before'/'after' all nodes in the tree.
