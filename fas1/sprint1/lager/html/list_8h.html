void list_destroy ( list_t *  list)

Destroys a list_t, freeing any memory allocated for the list.

Destroys the provided list and free's any memory allocated for it. Doesn't free any memory alloc'd for the elements stored within the list, they must be manually free'd be the user. Any use of the provided list_t is invalid after calling list_destroy.
